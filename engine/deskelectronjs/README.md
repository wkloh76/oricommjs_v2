# DeskelectronJS

## Objective

- This is desktop application engine module which use ElectronJS to interactive between devices OS physical resource and application.

## Startup

1. Apply code below to oricommjs_components coresetting.toml

2. electron-builer requierement Dependencies packages setting under devDependencies

   ```
    "devDependencies": {
      "electron": "40.3.0",
      "electron-builder": "26.7.0"
    }
   ```

3. electron/packager requierement Dependencies packages setting under devDependencies
   ```
    "devDependencies": {
      "@electron/packager": "19.0.3",
      "@electron/rebuild": "4.0.3",
      "electron": "40.3.0",
      "node-gyp": "12.2.0",
      "prebuild-install": "7.1.3"
    }
   ```
